## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- Yarn

### Setup

```bash
git clone https://github.com/hiromataba/repo_name.git
cd ./repo_name
```

### Install

```bash
yarn
```

### Usage

```bash
yarn start
```
